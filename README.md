# OROMEX - Oromia Ministry Past Exam App

A bilingual digital exam practice platform designed for students in Oromia, Dire Dawa, and Harar regions.

## Features

- **Bilingual Support**: Afan Oromo and Amharic language support
- **Regional Specific**: Content tailored for Oromia, Dire Dawa, and Harar regions
- **Grade Focused**: Specifically for Grades 7 and 8
- **Freemium Model**: Free access to Unit 1, premium for all content
- **Exam Security**: Anti-cheating measures and screen capture prevention
- **Offline Mode**: Premium users can download exams for offline practice
- **Progress Tracking**: Detailed analytics and performance history
- **Payment Integration**: Multiple local payment methods (CBE, Telebirr, Amole, HelloCash)

## Tech Stack

- **Frontend**: Flutter (Android, iOS, Web)
- **State Management**: Provider
- **Backend**: Node.js + Express.js
- **Database**: Firebase Firestore / MongoDB
- **Authentication**: Firebase Auth
- **Storage**: Firebase Storage
